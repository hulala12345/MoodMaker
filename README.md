# MoodMaker

MoodMaker is a simple web-based application that generates music playlists based on your current mood. It uses a small local song database and allows you to register an account, save playlists, and share them with others.

## Features
- Select a mood from the list or enter your own
- Generate a playlist matched to the mood
- Save playlists to your account
- Share playlists using a public link
- View your saved playlists

## Requirements
- Python 3.8+

## Running
Execute the following command and open `http://localhost:8000` in your browser:

```bash
python3 app.py
```
